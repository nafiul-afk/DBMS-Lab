#  Database Management Systems (DBMS) Lab

A structured hands-on lab course covering relational database design, SQL querying, and full-stack database project development.

---

##  Lab Schedule & Outcomes

| Lab | Topic / Assignment | CO | Learning Outcome |
|-----|-------------------|-----|-----------------|
| Lab 1 | Introduction to relational databases & working environment | CO2 | Familiarization with relational database systems and tools |
| Lab 2 | Implementation of relational database based on case study | CO2 | Design and implement a database from a real case study |
| Lab 3 | Manipulation of implemented relational databases | CO1 | Perform manipulation operations on a relational database |
| Lab 4 | Introduction to database project using DB server & web server | CO4 | Understand the difference between web/DB servers and initiate a project |
| Lab 5 | Basic database queries using SQL | CO1 | Write basic SQL queries with mathematical and logical operations |
| Lab 6 | Intermediate database queries using SQL | CO1 | Use joins, aggregation, and grouping in SQL |
| Lab 7 | Advanced database queries using SQL | CO1 | Write subqueries and use set operations in SQL |
| Lab 8 | Partial project presentation & group work evaluation | CO3, CO4 | Understand teamwork dynamics in project building |
| Lab 9 | Develop the project with database theories & techniques | CO4 | Formally develop final databases using learned techniques |
| Lab 10 | Project presentation & group work evaluation | CO3, CO4 | Evaluate teamwork and management for the project |
| Lab 11 | Final presentation of project using database & web server | CO3, CO4 | Apply security, integrity, and management hierarchies in databases |
| Lab 12 | Lab Exam | CO2 | Evaluate teamwork as well as project management skills |

---

## Tools & Technologies

- **Database:** MySQL 
- **SQL Client:** MySQL Workbench  
- **Language:** SQL, Python for web integration

---

## Getting Started

1. **Clone the repository**
   ```bash
   git clone https://github.com/your-username/dbms-lab.git
   cd dbms-lab
   ```

2. **Set up your database environment**
   - Install MySQL / PostgreSQL
   - Start your database server
   - Create a new database for each lab as instructed

3. **Run lab scripts**
   ```bash
   mysql -u root -p < Lab1/setup.sql
   ```

---

## 📌 Notes

- Each lab folder contains SQL scripts and a brief write-up of the tasks.
- Project labs (Lab 8–11) are group assignments — refer to your team's project folder.
- The lab exam (Lab 12) covers all topics from Lab 1–7.
